## Демо

[https://iarus.github.io/size-calc/public/](http://localhost:5000)


## Начало

Установить зависимости

```bash
cd size-calc
npm install
```


## Запустить dev сервер

```bash
npm run dev
```

Перейти по ссылке [localhost:5000](http://localhost:5000).


## Сборка приложения

```bash
npm run build
```
